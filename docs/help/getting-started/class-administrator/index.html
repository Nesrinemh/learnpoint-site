Kom igång som klassadministratör

Aktivera konto

Om det är första gången du loggar in i Learnpoint behöver du aktivera ditt konto:

  1. Gå till learnpoint.se/sign-in.html.
  2. Ange adressen till din skola.
  3. Klicka på Fortsätt.
  4. Nu kommer du till inloggningssidan för din skola.
  5. Klicka på Aktivera konto.
  6. Ange din e-postadress och klicka på OK.
  7. Inom kort får du ett mail med instruktioner för att aktivera kontot.

Logga in

När du aktiverat ditt konto loggar du in på följande sätt:

  1. Gå till learnpoint.se/sign-in.html.
  2. Ange adressen till din skola.
  3. Klicka på Fortsätt
  4. Ange användarnamn och lösenord.
  5. Klicka på Logga in.

Klasser och kurser

De aktuella klasser som du är registrerade som klassadministratör för visas i menyn till vänster under rubriken grupper. Väljer du en klass kan du se och gå till klassens kurser via fliken Kurser. Trycker du på Grupper i vänstermenyn så visas en lista på alla grupper som du är kopplad till. Du kan sedan se och hantera klassen eller kursens nyheter, material, studerande och personal.

Hantera studenter i klasser

Du kan lägga till studerande i klassen. Om skolan har använt yh-antagning.se för antagning till en klass så kan du skriva in studerande direkt från yh-antagning.se. Du kan också ändra status på inskrivna studerande när de gör studieupphåll, studieavbrott eller helt enkelt inte dyker upp när utbildningen börjar.

Redigera studenter

Varje person i Learnpoint har en profilbild och den som är administratör kan byta profilbilder på de studerande i klassen genom att ladda upp en bildfil eller genom att använda datorn eller mobilens webbkamera. Du kan även ändra kontaktuppgifter för de studerande i dina klasser.
