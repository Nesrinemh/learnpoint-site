New feature: Import course content Ny funktion: Importera kursinnehåll

Dhanfia Rahman

October 5, 2023 5 Oktober, 2023

Learnpoint has been updated with a new feature: Import course content. This feature is useful when you want to reuse content instead of recreating it from scratch. Learnpoint har uppdaterats med en ny funktion: Importera kursinnehåll. Funktionen är användbar när du vill återanvända befintligt innehåll, istället för att skriva allt innehåll från början.

You will find the feature at the bottom of the content page: Funktionen finns längst ned på innehåll-sidan:

What can you import? Vad kan importeras?

You can import course items from Learnpoint courses. You can select specific items to import — or choose all of them. Note that it's only the content and attachments that will be imported, not the item settings. Du kan importera kursdelar från Learnpoint-kurser. Du kan importera specifika kursdelar eller välja allihop. Notera att det endast är innehållet och bilagorna som importeras, inte kursdelarnas inställningar.

Who can import from a course? Vem kan importera från en kurs?

You can always import content from all your own courses. But each course also has a setting that, when enabled, will allow every teacher in the school to import content from that course. Du kan alltid importera innehåll från dina egna kurser. Men varje kurs har dessutom en inställning, och om den inställningen är aktiverad, så kan alla lärare på skolan importera innehåll från den kursen.

For existing courses, the setting has to be manually enabled, but when you create a new course, it will be enabled by default. För befintliga kurser behöver inställningen aktiveras manuellt, men när du skapar en ny kurs så är inställningen aktiverad automatiskt.
