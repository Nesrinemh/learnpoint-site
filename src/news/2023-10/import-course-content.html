Dhanfia Rahman

October 5, 2023 5 Oktober, 2023

Learnpoint has been updated with a new feature: Import course content. This feature is useful when you want to reuse content instead of recreating it from scratch.

You will find the feature at the bottom of the content page:

What can you import?

You can import course items from Learnpoint courses. You can select specific items to import — or choose all of them. Note that it's only the content and attachments that will be imported, not the item settings.

Import setting | Who can import content from a course?

You can import content from all your own courses. But each course also has a setting that, when enabled, will allow every teacher in the school to import content from that course.

For existing courses, the setting has to be manually enabled, but when you create a new course, it will be enabled by default.
